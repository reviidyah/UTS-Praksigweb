Mahasiswa Teknik Geodesi Universitas Gadjah Mada dengan minat dalam pemetaan dan analisis data geospasial. Memiliki pengalaman magang di Bendungan Karangnongko dengan fokus pada bidang pemetaan. Kegiatan Praktikum SIG WEB yang Telah Dilaksanakan Setiap Minggu.

Minggu 1, 2, dan 3

Pada minggu pertama, kedua, dan ketiga Praktikum SIG WEB, dilakukan pembuatan halaman HTML sederhana yang berisi elemen-elemen seperti gambar, teks yang terhubung ke link eksternal, tulisan miring, tulisan berwarna, tulisan tebal, tabel, dan form. Pada pembuatan website ini juga menggunakan css untuk mengatur tampilan web, selain itu juga dilakukan pembuatan CV menggunakan website builder wix (wix.com) dan dipublish serta memperoleh url dari CV yang telah dibuat sehingga CV dapat diakses oleh siapa saja.

Minggu 4

Pada minggu keempat Praktikum SIGWEB, dilakukan pembuatan halaman web sederhana dengan lima soal.

  • Membuat program untuk menghitung faktorial dari input pengguna.
  • Validasi input password dengan syarat tertentu.
  • Validasi nama dan umur yang diinput.
  • Membuat program utnuk menghitung bilangan prima.
  • Membuat program untuk menghitung bilangan Fibonacci
Minggu 5

Pada minggu kelima Praktikum SIG WEB, dilakukan pembuatan Curriculum Vitae (CV) menggunakan template BootStrap. Selain itu, membuat akun github dan repository yang berisi CV, kemudian menggunakan Git utnuk melakukan clone repository dan dan menambahkan pekerjaan ke dalam repository Github. Pada minggu ini juga dibuat Landing Page sebagai tampilan awal dari website.

Minggu 6

Pada minggu keenam Praktikum SIG WEB, dilakukan pembuatan WebMap sederhana dengan menggunakan Leaflet. WebMap ini menampilkan poin/marker lokasi dan pop-up yang berisi informasi lokasi, seperti lokasi Departemen Teknik Geodesi, Kos, FKKMK, dan lain-lain. Pada minggu ini juga ditambahkan rute jalan dari kos ke Departemen Teknik Geodesi menggunakan leaflet routing machine.

Minggu 7

Pada minggu ketujuh praktikum SIGWEB, melanjutkan WebMap dari minggu sebelumnya dengan menambahkan fitur tambahan seperti data GEOJSON yang berisi batas wilayah administrasi DIY. Selain menambahkan GEOJSON ke dalam WebMap, juga menambahkan fitur untuk memilih basemap yang digunakan, fitur pencarian, dan tombol zoom in zoom out.
